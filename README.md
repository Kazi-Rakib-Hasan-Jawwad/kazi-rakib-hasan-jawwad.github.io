# Kazi Rakib Hasan Academic Website

This repository contains a simple static academic website for GitHub Pages.

Target URL:

```text
https://kazi-rakib-hasan-jawwad.github.io
```

## Setup

1. Create a GitHub repository named exactly:

```text
kazi-rakib-hasan-jawwad.github.io
```

2. Upload these files to the repository root:

```text
index.html
style.css
README.md
```

3. Go to repository **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**.
6. Save and wait for GitHub Pages to publish.

## Edit content

Update `index.html` to add more publications, DOI links, projects, CV link, or Google Scholar link.
